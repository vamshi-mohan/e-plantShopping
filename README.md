e-plantShopping is a React-based shopping cart application for an online plant store called Paradise Nursery.  
The application allows users to browse different categories of house plants, add plants to a shopping cart, manage quantities, and view the total cost dynamically.

## Features
- Landing page with Paradise Nursery branding and Get Started button
- Product listing page with multiple plant categories
- Add to Cart functionality with Redux state management
- Shopping cart with quantity control and delete option
- Dynamic cart item count and total price update
- Navigation bar with Home, Plants, and Cart links

## Technologies Used
- React
- Redux Toolkit
- JavaScript (ES6)
- HTML5
- CSS3
